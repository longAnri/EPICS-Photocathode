这是一个基于EPICS的光阴极制备设备控制程序，其中程序我们均已测试过，大家可以下载参考。

MQTT-Client文件是已经写好的MQTT客户端文件，可根据需求添加过量变量发布至服务器端，但需要修改服务id。

StatusMonitorApp文件里包含大量前期编写的测试文档，这些文件不一定可以使用，仅作参考

ChecksumConverter文件里包含了修改过的ChecksumConverter.cc，其中支持SQM160的crc校验方式

各个应用程序中包含多个OPI界面文件，仅作参考，可根据自身需求调用相关过程变量PV

最后申明，有些应用程序是直接引用其他github的程序，如IPCMiniApp和KEythley6485App。非常感谢他们的分享。本项目文件也是公开的，也欢迎各位的批评指正。
